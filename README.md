# Welcome to Junsong Cang's public research archive!
This repo contains my research codes for each publication, you can use these codes to reproduce results in my paper.
Some data files might not be included if they are too large but they are available upon request to cangjunsong@outlook.com or cangjunsong@icloud.com
Each publication has its own folder with a detailed README.md inside, folders can be named with brief summary of the paper. Currently the repo contains codes for following publications:

- DM_21cm_2025_PRD
    Paper: Signatures of inhomogeneous dark matter annihilation on 21-cm
    DOI: 10.1103/69jk-m5xg
    arXiv: 2312.17499

More will be added
